# Problem: Reverse String

## Description
Write a function that reverses a string.  
The input string is given as an array of characters and must be reversed **in-place**.

## Pattern
Two Pointers / Strings

## Difficulty
Easy

## Approach
1. Use two pointers: `left` at the start and `right` at the end.
2. Swap the characters at both pointers.
3. Move the pointers toward the center.
4. Continue until the entire string is reversed.

## Time Complexity
O(n)

## Space Complexity
O(1)

## Platform
LeetCode
