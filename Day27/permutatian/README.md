## Description
Given an array of distinct integers `nums`, return all possible permutations.

## Pattern
Backtracking / Recursion

## Difficulty
Medium

## Approach
1. Use backtracking to build permutations.
2. Track used elements to avoid repetition.
3. Add a permutation to the result when its length equals the input size.
4. Backtrack to explore other permutations.

## Time Complexity
O(n!)

## Space Complexity
O(n)
