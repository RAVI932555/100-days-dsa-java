# Problem: 4Sum

## Description
Given an array of integers, find all unique quadruplets that sum up to a given target value.

### Pattern
Sorting + Two Pointers

### Difficulty
Medium

### Approach
1. Sort the array.
2. Fix two indices using nested loops.
3. Use two pointers to find remaining two values.
4. Skip duplicates to ensure unique quadruplets.
