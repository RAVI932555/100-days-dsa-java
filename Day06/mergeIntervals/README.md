# Problem: Merge Intervals

## Description
Given an array of intervals, merge all overlapping intervals and return the non-overlapping intervals.

### Pattern
Sorting / Intervals

### Difficulty
Medium

### Approach
1. Sort intervals based on start time.
2. Iterate and merge overlapping intervals.
3. Add merged intervals to the result list.

### Time Complexity
O(n log n)

### Space Complexity
O(n)

### Platform
LeetCode
