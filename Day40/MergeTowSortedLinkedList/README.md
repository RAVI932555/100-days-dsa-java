## Description
You are given the heads of two sorted linked lists `list1` and `list2`.  
Merge the two lists into one sorted linked list and return the head of the merged list.

## Pattern
Two Pointers / Linked List

## Difficulty
Easy

## Approach
1. Create a dummy node to simplify merging.
2. Use two pointers to traverse both lists.
3. Compare current nodes and attach the smaller one to the merged list.
4. Move the pointer of the selected list.
5. Attach remaining nodes once one list is exhausted.
