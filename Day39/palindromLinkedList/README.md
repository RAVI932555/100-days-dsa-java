## Description
Given the head of a singly linked list, return `true` if it is a palindrome, otherwise return `false`.

## Pattern
Two Pointers / Linked List Manipulation

## Difficulty
Easy

## Approach
1. Use slow and fast pointers to find the middle of the list.
2. Reverse the second half of the linked list.
3. Compare nodes from the first half and the reversed second half.
4. If all values match, the list is a palindrome.
