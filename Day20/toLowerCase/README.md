## Description
Given a string `s`, return the string after replacing every uppercase letter with its lowercase equivalent.

## Pattern
String / Character Manipulation

## Difficulty
Easy

## Approach
1. Traverse each character in the string.
2. If the character is between `'A'` and `'Z'`, convert it to lowercase by adjusting its ASCII value.
3. Append the character to the result string.
4. Return the final string.

## Time Complexity
O(n)

## Space Complexity
O(n)
