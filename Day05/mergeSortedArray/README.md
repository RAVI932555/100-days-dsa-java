# Problem: Merge Sorted Array

## Description
Given two sorted integer arrays `nums1` and `nums2`, merge `nums2` into `nums1` as one sorted array.

### Pattern
Two Pointers / Array Manipulation

### Difficulty
Easy

### Approach
1. Start merging from the end of both arrays.
2. Compare elements and place the larger one at the end.
3. Continue until all elements are merged.

### Time Complexity
O(n + m)

### Space Complexity
O(1)

### Platform
LeetCode
