## Description
Reverse bits of a given 32-bit unsigned integer and return the resulting integer.

## Pattern
Bit Manipulation

## Difficulty
Easy

## Approach
1. Initialize a result variable to 0.
2. Iterate 32 times.
3. Shift result left and add the last bit of the number.
4. Shift the number right in each iteration.
5. Return the reversed result.

## Time Complexity
O(1)

## Space Complexity
O(1)
