# 🔤 First Unique Character in a String

## 📖 Problem

Given a string `s`, find the index of the **first non-repeating character**.
If none exists, return `-1`.

---

## 🧠 Approach (Frequency Count)

### Steps:

1. Create a frequency array/map
2. Count occurrences of each character
3. Traverse string again:

   * Return index where frequency == 1
