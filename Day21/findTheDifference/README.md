## Description
You are given two strings `s` and `t`.  
String `t` is generated by shuffling string `s` and adding one extra character.

Return the extra character.

## Pattern
Bit Manipulation / Hashing

## Difficulty
Easy

## Approach
1. Initialize a variable to store XOR result.
2. XOR all characters in string `s`.
3. XOR all characters in string `t`.
4. The remaining value represents the extra character.

## Time Complexity
O(n)

## Space Complexity
O(1)
