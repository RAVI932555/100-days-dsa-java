## Description
Given a string `s`, determine if it is a palindrome after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters.

## Pattern
Two Pointers / String Processing

## Difficulty
Easy

## Approach
1. Use two pointers: one at the start and one at the end of the string.
2. Skip non-alphanumeric characters.
3. Compare characters after converting them to lowercase.
4. If all characters match, return `true`.

## Time Complexity
O(n)

## Space Complexity
O(1)
