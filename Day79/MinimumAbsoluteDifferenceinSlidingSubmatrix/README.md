# 🧮 Minimum Absolute Difference in Sliding Submatrix

## 📖 Problem

Given a matrix and a window size `k × k`,
find the **minimum absolute difference between any two elements** inside each submatrix.

---

## 🧠 Approach

1. Slide a window of size `k × k` over the matrix.
2. For each submatrix:

   * Extract elements
   * Sort them
   * Compute minimum difference between adjacent elements
3. Track the global minimum.

Optimized approach:

* Use data structures like **TreeSet / Balanced BST** to maintain sorted order dynamically.

---

## 💡 Key Concepts

* Sliding Window (2D)
* Sorting
* Matrix traversal
* Optimization using TreeSet

---

## ⏱ Time Complexity

Naive: O(n² × k² log k²)
Optimized: Better with balanced structures

## 💾 Space Complexity

O(k²)

---

## 🚀 Example

Input:
Matrix + window size

Output:
Minimum absolute difference among all submatrices
