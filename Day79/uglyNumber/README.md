# 🔢 Ugly Number

## 📖 Problem

An ugly number is a positive integer whose prime factors are limited to **2, 3, and 5**.

Given an integer `n`, return `true` if it is an ugly number.

---

## 🧠 Approach

Steps:

1. If `n <= 0`, return false.
2. Continuously divide `n` by:

   * 2
   * 3
   * 5
3. If after division `n == 1`, then it is an ugly number.

---

## 💡 Example

Input

```
n = 6
```

Output

```
true
```

Explanation
6 = 2 × 3 → only allowed primes.

---

## ⏱ Time Complexity

O(log n)

## 💾 Space Complexity

O(1)

---

## 🔑 Concepts Used

* Number theory
* Prime factorization
* Iterative reduction
