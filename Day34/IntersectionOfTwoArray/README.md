## Description
Given two integer arrays `nums1` and `nums2`, return an array of their intersection.  
Each element in the result must be **unique**, and the result can be returned in any order.

## Pattern
Hash Set

## Difficulty
Easy

## Approach
1. Store all elements of the first array in a `HashSet`.
2. Traverse the second array and check for common elements.
3. Add common elements to a result set to ensure uniqueness.
4. Convert the result set to an array.
