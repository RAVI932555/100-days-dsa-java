## Description
Given the head of a singly linked list, sort the list using **insertion sort** and return the sorted list.

## Pattern
Linked List Manipulation / Sorting

## Difficulty
Medium

## Approach
1. Use a dummy node to build the sorted portion of the list.
2. Traverse the original list node by node.
3. For each node, find the correct position in the sorted list.
4. Insert the node into the sorted list.
5. Continue until all nodes are processed.
