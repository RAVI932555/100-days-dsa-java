# Problem: Decimal to Integer

## Description
Given a decimal number (or numeric string),  
convert it into its corresponding integer value.

## Pattern
Math / Strings

## Difficulty
Easy

## Approach
1. Initialize a result variable.
2. Traverse each digit of the decimal number.
3. Multiply the result by 10 and add the current digit.
4. Continue until all digits are processed.
5. Return the final integer value.

## Time Complexity
O(n)

## Space Complexity
O(1)

## Platform
General / LeetCode-style
