# Problem: Add Binary

## Description
Given two binary strings `a` and `b`,  
return their sum as a **binary string**.

## Pattern
Math / Strings

## Difficulty
Easy

## Approach
1. Use two pointers starting from the end of both strings.
2. Add corresponding bits along with carry.
3. Append the result bit to a StringBuilder.
4. Reverse the result to get the final binary string.

## Time Complexity
O(n)

## Space Complexity
O(n)

LeetCode
