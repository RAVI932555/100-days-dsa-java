# Problem: Find Missing and Repeated Values

## Description
Given a 2D grid representing numbers from 1 to n², one number is missing and one is repeated. Find both values.

### Pattern
Arrays / Math

### Difficulty
Easy

### Approach
- Use frequency counting or mathematical sum approach.
- Identify the number with frequency > 1 (repeated).
- Identify the missing number.

### Time Complexity
O(n²)

### Space Complexity
O(n²) or O(1) (math-based)

### Platform
LeetCode
